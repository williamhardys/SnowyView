# Project Title

Add a very very simple snow effect to your views



### Installing

You only need to drag `SnowyView.swift` to your project folder

And then suprise 
```
override func viewDidLoad() {
        super.viewDidLoad()
        
        self.view.AddSnow(Interval: 0.5)
        
 }
```

And Run the App
